# Chinese Project

A comprehensive application for learning Chinese, featuring a backend API, frontend user application, and an admin portal.

## Structure

- **apps/backend-api**: NestJS backend application.
- **apps/frontend**: React/Next.js frontend application.
- **apps/admin**: Admin portal.
- **packages/**: Shared libraries and types.

## Getting Started

Please refer to the individual `README.md` files in each application directory for specific setup instructions.
